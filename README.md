# Web-Coffe-Nhom2
Phiên bản dự án học môn Thực hành làm việc nhóm của Thầy Vũ Sơn Lâm.
Project version of Teamwork Practice subject by Mr. Vu Son Lam.

-Cài đặt môi trường và chạy unit test.
# Khởi tạo dự án Node.js (nếu chưa có)
npm init -y
# Cài đặt Jest và môi trường DOM giả lập
npm install --save-dev jest jest-environment-jsdom
# Chạy 
npm test
